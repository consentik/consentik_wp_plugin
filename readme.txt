== Installation ==
= Automatic installation =

1. Log in to your WordPress dashboard.
2. Go to "Plugins" > "Add New".
3. Search for "Consentik".
4. Click "Install Now" and then click "Activate".
5. Go to the Consentik settings page in your WordPress admin.
6. Register an account at https://cmp.consentik.com/admin/register to obtain your Site ID and Instance ID.
7. Enter your Site ID and Instance ID in the plugin settings and save changes.

= Manual installation =

1. Download the plugin ZIP file.
2. Upload the plugin ZIP file via "Plugins" > "Add New" > "Upload Plugin", then click "Install Now".
   Alternatively, unzip the file and upload the folder to the `/wp-content/plugins/` directory (so it becomes `/wp-content/plugins/consentik-cmp/`).
3. Activate the plugin through the "Plugins" menu in WordPress.
4. Go to the Consentik settings page in your WordPress admin.
5. Register an account at https://cmp.consentik.com/admin/register to obtain your Site ID and Instance ID.
6. Enter your Site ID and Instance ID in the plugin settings and save changes.

== Frequently Asked Questions ==

= Is Consentik compatible with Google Consent Mode V2? =
Yes, Consentik is fully integrated with Google Consent Mode V2 (both Basic and Advanced implementations). It automatically signals user consent status to Google tags (Ads, Analytics) to help recover conversion data while respecting user privacy.

= How do I get my Site ID and Instance ID? =
After installing the plugin, you need to register a free account at [https://cmp.consentik.com/admin/register](https://cmp.consentik.com/admin/register). Once logged in, add your website to the dashboard, and you will be provided with the Site ID and Instance ID required to activate the plugin.

= Does this plugin block cookies automatically? =
Yes. Consentik includes an advanced scanner that categorizes cookies and scripts. It ensures that non-essential scripts (like marketing or analytics cookies) are blocked until the visitor explicitly grants consent.

= Is Consentik compliant with IAB TCF v2.2? =
Absolutely. We support the IAB Transparency and Consent Framework (TCF) v2.2. This is essential for publishers monetizing with Google AdSense, Ad Manager, or other ad networks that require TCF compliance to serve personalized ads in Europe.

= Can I customize the design of the cookie banner? =
Yes, you can fully customize the banner's appearance, including colors, fonts, text content, and position (top/bottom/center) directly from the plugin settings to match your website's branding.

= Which privacy regulations does Consentik support? =
Consentik is designed to help you comply with major global privacy laws, including GDPR (Europe), CCPA/CPRA (California), LGPD (Brazil), PDPE (Singapore), and AAPI (Japan).

= Do I need coding skills to use this plugin? =
No coding is required. The setup is designed to be user-friendly. Simply install the plugin, enter your IDs, and configure your preferences using the visual settings panel.
